	LiquidCrystal::noDisplay


	void LiquidCrystal::noDisplay();

This function turns the LCD's display off.

Usage:

	LiquidCrystal lcd(8, 9, 4, 5, 6, 7);
	lcd.begin(16, 2);
	lcd.noDisplay();
